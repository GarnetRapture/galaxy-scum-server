## 게임 시작하기 (0단계)

### 인증 및 서버 선택
1. Steam에서 SCUM 구매 및 설치
2. 게임 시작
3. 서버 목록에서 "갤럭시" 또는 "KOR_PVE" 검색
4. 방제: "KOR_PVE 갤럭시 since2021 경험치이벤트[디코상점-웰컴팩-직장인뉴비환영]"

- 스폰 위치 선택 (일반적으로 해변이나 도시 권장)

## 1단계: 첫 30분 생존 (Critical)

> **검증상태**: partial  
> **sourceIds**: `source-scum-steam-store`  
> **주의사항**: 물 끓이기 시간, 질병 감염 경로 구체화 필요

### 당신의 상태 확인
게임 시작 후 **즉시 확인해야 할 것**:
- **배고픔 게이지**: 화면 하단 좌측 (음식 아이콘)
- **목마름 게이지**: 화면 하단 좌측 (물 아이콘)
- **피로도 게이지**: 화면 하단 좌측 (별 아이콘)

### 긴급 우선순위
1. **깨끗한 물 찾기** (가장 중요)
   - 호수나 강 근처로 이동
   - 하천수를 마실 수 있으나 감염 위험
   - 물통(Water container) 찾아서 담기
   - 물을 끓여서 (캠프파이어 필요) 안전하게 마시기

2. **음식 찾기** (두 번째 우선순위)
   - 농가 주변에 음식 있음 (곡물, 채소)
   - 동물 시체가 있으면 고기 채취 (나이프 필요)
   - 베리나 과일 채집
   - 먹을 것을 찾을 때까지 이동하기

3. **피난처 찾기** (첫날 밤 전)
   - 버려진 집이나 건물 찾기
   - 폐쇄된 공간이 좋음 (좀비로부터 안전)
   - 불을 킬 재료 모으기 (막대, 종이)

### 첫 밤 버티기
- 집 안에서 저장(Save) → 밤 동안 체력 회복
- 불이 있으면 더 빨리 회복
- 문을 닫아서 좀비 침입 방지

## 3단계: 초급 기술 습득 (Days 3-7)

> **sourceIds**: `source-scum-steam-store`  
> **주의사항**: 스킬 경험치 배수, 레벨업 시간 needs-review

### 스킬 이해하기
게임의 모든 행동은 **스킬 경험치 획득**:
- **Crafting**: 도구나 아이템을 만들 때
- **Cooking**: 음식을 조리할 때
- **Melee Combat**: 근접 공격할 때
- **Ranged Combat**: 총이나 활로 공격할 때
- **Hunting**: 동물을 사냥할 때
- **Fishing**: 낚시할 때
- **Farming**: 경작할 때

스킬 레벨이 올라갈수록 더 나은 아이템을 만들 수 있습니다.

### 첫 번째 도구 만들기
**돌 도끼** (가장 기본):
- 재료: 나뭇가지 × 2, 돌 × 1
- 방법: 인벤토리에서 제작 메뉴 열기
- 효용: 나무 벌채, 좀비 처치

### 기본 집 짓기
1. **위치 선택**: 좋아하는 곳 (안전하고 물 가까운 곳 추천)
2. **기초(Foundation)** 놓기 (목재 필요)
3. **벽(Wall)** 세우기 (목재 필요)
4. **문(Door)** 달기 (안전성 증가)
5. **창(Window)** 만들기 (채광)
6. **침대(Bed)** 만들기 (저장점 설정)

## 4단계: 생존의 깊이 (Week 2+)

> **검증상태**: partial  
> **sourceIds**: `source-scum-steam-store`  
> **주의사항**: 신진대사 수치/계산식, 회복 시간, 질병 감염 시간 needs-review

### 신진대사 시스템 이해
**네 가지 핵심 게이지**:
1. **Hunger (배고픔)**: 음식 섭취로 해결
2. **Thirst (목마름)**: 깨끗한 물 섭취로 해결
3. **Fatigue (피로)**: 자면서 회복
4. **Health (체력)**: 부상 치료로 회복

각 게이지가 낮아지면 최대 체력이 감소합니다.

### 부상 및 질병 관리
**흔한 부상**:
- Cut (상처): 붕대나 천으로 치료
- Fracture (골절): 부목 필요
- Burn (화상): 알로에나 의약품
- Infection (감염): 항생제나 항염증제

**질병**:
- Cholera (콜레라): 오염된 물/음식 → 항생제 필요
- Flu (독감): 추운 환경 → 자체 치유 또는 의약품

### 루팅 효율화
**효율적인 루팅 위치**:
- 농가: 음식, 도구
- 군사 기지: 무기, 탄약 (위험)
- 도시 건물: 의약품, 전자제품
- 차고: 차량 부품, 도구

**루팅 팁**:
- 큰 백팩 가져가기 (용량 증가)
- 밤에 루팅하지 않기 (좀비 더 활동)
- 항상 물러날 경로 확보

### 무기 선택
**초급자 추천**:
- **근접무기** (도끼, 검): 조용함, 연료 불필요, 근거리만 가능
- **활** (Bow): 조용하고 효율적, 화살 만들어야 함
- **총기** (총): 시끄러움 (좀비 유인), 탄약 필요, 강력함

> **주의사항**: INTO THE WILD 랜드 동물 행동 needs-review

### 차량 이용 (갤럭시 서버)
**갤럭시 서버 무료 차량 렌트 정책**:
1. 디스코드에서 관리자(블락)에게 요청
2. 무료 차량 제공 (신규 유저 환영)
3. 새 차량을 구매할 때만 기존 차량 반납
4. 자연 차량 스폰 없음 (렌트만 가능)

**차량 용도**:
- 빠른 이동
- 많은 자원 운반
- 루팅 지역 접근

### 사냥 및 요리
**기본 사냥**:
1. 동물 찾기 (멧돼지, 사슴 등)
2. 무기 준비 (활이나 총)
3. 바람 방향 주의 (냄새로 감지)
4. 은폐하면서 접근
5. 공격 및 해체

**요리**:
- 고기 → 구워서 영양가 증가
- 채소 + 고기 = 스튜 (더 많은 영양)
- INTO THE WILD 즉흥 요리 (새로운 레시피 가능)

### 농사 시작
1. 경작지 만들기 (목재 필요)
2. 씨앗 심기 (감자, 옥수수, 토마토 등)
3. 매일 물주기
4. 약 2-3일 후 수확
